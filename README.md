 Internet Speed Monitor & Logger

A Python tool that automatically tests your **internet speed** (ping, download, upload)
at fixed intervals and logs results to a CSV file. It also generates a graph to visualize
speed trends over time.

---
 Features
- Runs internet speed tests using `speedtest-cli`
- Logs results (timestamp, ping, download, upload)
- Visualizes data using matplotlib
- Customizable interval and test count
